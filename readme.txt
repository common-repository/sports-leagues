== Description ==
Full-featured solution for any team sports website. Manage players, games, teams, rosters, tournaments, standings and leagues with ease.

Suitable for &#127945; rugby, &#127936; basketball, &#127952; volleyball, handball, &#127951; cricket, &#9918; baseball, &#127954; ice hockey, water polo, and most of other team sports.

&#9193; For Football(Soccer) &#9917; use my other plugin - [Football Leagues](https://wordpress.org/plugins/football-leagues-by-anwppro/)

Limitation: Sports Leagues plugin can only be used for ONE sport at a time.

== Installation ==

= Minimum Requirements =

* PHP version 5.6 or greater
* WordPress 4.7+
* Enabled WordPress Rest API

= Installation =
1. Go to Plugins >> Add new
1. Type "Sports Leagues" in Search Plugins box.
1. Click Install Now to install the "Sports Leagues".
1. If successful, click Activate Plugin to activate it.
1. Install required CMB2 plugin. [CMB2](https://wordpress.org/plugins/cmb2/ "CMB2")

== Frequently Asked Questions ==

= Will Sports Leagues plugin work with my theme? =

The plugin works with most classic themes out of the box, but Full Site Editing (FSE) themes are not fully supported. If you have any CSS styling issues feel free to ask for help.
